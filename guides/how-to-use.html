熊狩シミュレーターの使い方

初めての方向けに、ゲーム内のどの画面を見て何を入力すればよいかを順番に解説します。所要時間は約1分です。

  1. STEP1: あなたのステータスを登録する。ゲーム画面左上の領主アイコンをタップし、プロフィール内の「ステータス」を開きます。「部隊攻撃力」「部隊殺傷力」と、「盾兵攻撃力」「盾兵殺傷力」など兵種ごとの%が表示されているので、その数値をそのまま入力してください。研究・領主装備・チャーム・ペットなどの合算値が表示されているため、個別に足し算する必要はありません。兵種Tierは、いま熊狩に出している兵士の段階(T10、FC1など)を選びます。
  2. STEP2: 集結主の英雄を選ぶ。まず「現在の世代」をサーバーの最新世代に合わせます(これより後の世代の英雄は選択肢に出ません)。盾・槍・弓の3枠をタップして、集結を立てる人が使う英雄を選びます。カード右上でスキルレベル(1〜5)、専用装備を持つ英雄は専用装備レベル(0〜10)を設定できます。枠の色はレアリティ(青=R、紫=SR、金=SSR)を表します。
  3. STEP3: 参加者の英雄を4人選ぶ。集結に乗るメンバーのうち、バフ枠に採用される4人の「左端英雄」を選びます。与ダメ系・攻撃系・殺傷系・確率系など、スキルの種類をバラけさせると掛け算で伸びます。第1スキルが熊に効かない英雄は「熊スキルなし」と表示されます。
  4. STEP4: 兵士の数と比率、シリル天賦を設定する。兵士の合計数を入力し、比率プリセット(10:30:60など)をタップすると自動で振り分けられます。自分の定番比率は「＋この比率を登録」で保存できます。シリルを育てている方は天賦「狩人の心得」のレベルを、罠の強化が最大でない同盟は罠強化ボーナスの%を設定してください。

結果の見方

  • 予測ダメージポイント — 1ラリーあたりの期待値です。
  • ダメージ幅 — 確率発動スキル(50%で発動など)を選んでいる場合、全部不発のときの下限と全部発動のときの上限を表示します。
  • つぎに強化すると伸びるもの — 現在の設定を基準に「その1手で何%伸びるか」を自動計算したランキングです。育成の優先順位付けに使えます。
🎯
精度を上げたい場合(キャリブレーション):「実測キャリブレーション」を開き、現在の設定と同じ条件で実際に行ったラリー1回分のダメージを入力して「自動補正する」を押してください。推定値と実測の比率から校正係数が計算され、以降の表示があなたの環境に合った絶対値になります。装備やペットを大きく更新したら再校正をおすすめします。

入力でつまずきやすいポイント

正確な結果を出すために、入力時によくある疑問を補足します。

  • ステータスは「%表示」のものを入れる — プロフィールの「部隊攻撃力」などは%で表示されています。実数ではなくその%の数字をそのまま入れてください(例:120%なら「120」)。0の項目は0と入力します。
  • TierとFC(火結晶)は別々に選ぶ — 兵種Tier(T1〜11)と火結晶レベル(FC0〜10)は独立した別軸です。たとえば「T10だがFC8まで上げている」「FC10だがT11はまだ」というケースも正しく入力できます。Tierとは別にFCも忘れず設定しましょう。
  • ペット・バフアイテムは使うものだけ — ホラアナライオン(攻撃)、サーベルタイガー(殺傷)、巨大カメレオン(敵防御ダウン)、2時間ブースト(攻撃/殺傷/防御ダウン)は、実際に使うものだけ入力します。使わない欄は0のままでOKです。
  • 専用装備Lvは英雄カードで設定 — 集結主の各英雄カードに専用装備Lv(0〜10)の欄があります。世代とLvに応じてステータス上昇が自動反映されます。

なぜ実測とズレることがあるのか

このツールは、公開仕様とコミュニティ検証に基づく「制約付き推定」モデルで動いています。ゲーム内部の正確な計算式は公開されていないため、補正なしの素の値は実際のポイントとズレることがあります。これは仕様であり、バグではありません。

だからこそ用意しているのが実測キャリブレーションです。あなたの実測ポイントを1回入力すれば、ツールが自動で補正係数Cを算出し、以降はあなたの環境に合った絶対値で表示されます。一度校正すれば、英雄やペット、比率を変えたときの「相対的な伸び」を正確に比較できるようになります。まずは1回、実測値で校正してから使うのがおすすめです。

How to use the Bear Hunt Simulator

For first-timers: a step-by-step guide to which in-game screens to check and what to enter. Takes about 1 minute.

  1. STEP 1: Enter your stats. Tap the lord icon (top-left), open "Stats" in your profile. You'll see "Army Attack", "Army Lethality" and per-type values like "Infantry Attack/Lethality". Enter those numbers as-is — they already include research, gear, charms and pets, so no manual adding is needed. For troop Tier, pick the tier of the troops you currently send to Bear Hunt (T10, FC1, etc.).
  2. STEP 2: Pick the rally leader's heroes. First set "Current generation" to your server's latest (heroes newer than this won't appear). Tap the Inf/Lancer/Mks slots to pick the heroes the leader uses. Set skill level (1-5) at the top-right of each card, and gear level (0-10) for heroes with exclusive gear. Frame color shows rarity (blue=R, purple=SR, gold=SSR).
  3. STEP 3: Pick 4 joiner heroes. Choose the 4 "left heroes" of joiners that get buff slots. Spreading skill types (DMG / ATK / Lethality / proc) multiplies your output. Heroes whose first skill doesn't work on the bear show "No bear skill".
  4. STEP 4: Set troops, ratio and the Cyrille talent. Enter total troops, then tap a ratio preset (e.g. 10:30:60) to auto-split. Save your go-to ratio with "+ Save this ratio". If you've leveled Cyrille, set the "Hunter's Heart" talent level; if your alliance's trap isn't maxed, set the trap upgrade bonus %.

Reading the result

  • Predicted damage points — the average for one rally.
  • Damage range — if you picked probability skills (e.g. 50% chance), it shows the floor (all miss) and ceiling (all proc).
  • What to upgrade next — a ranking auto-computed from your current setup showing how much each single move raises damage. Use it to prioritize.
🎯
For more accuracy (calibration): open "Real-measure calibration", enter the damage from one real rally done with the same setup, and press "Auto-calibrate". A calibration factor is computed from the ratio of estimate to measurement, so later figures become absolute values fitted to your account. Re-calibrate after big gear or pet updates.

Common input pitfalls

To get accurate results, here are some common input questions.

  • Enter the "%" values — profile values like "Army Attack" are shown as percentages. Enter that % number as-is (e.g. 120 for 120%), not a raw value. Enter 0 for any zero field.
  • Select Tier and FC (Fire Crystal) separately — troop Tier (T1-11) and Fire Crystal level (FC0-10) are independent axes. Cases like "T10 but raised to FC8" or "FC10 but no T11 yet" are entered correctly. Don't forget to set FC separately from Tier.
  • Only the pets/buff items you use — Cave Lion (attack), Saber-tooth Tiger (lethality), Giant Chameleon (enemy defense down), and 2-hour boosts (attack/lethality/defense-down) — only enter the ones you actually use. Leave unused fields at 0.
  • Set exclusive gear Lv on the hero card — each leader hero card has an exclusive-gear Lv field (0-10). Stat boosts apply automatically based on generation and level.

Why it can differ from your actual points

This tool runs on a "constraint-based estimate" model from public specs and community verification. Since the game's exact internal formula isn't published, the raw (uncorrected) value can differ from your actual points. This is by design, not a bug.

That's exactly what real-measure calibration is for. Enter your measured points once and the tool auto-computes factor C, then shows absolute values fitted to your account. After calibrating once, you can accurately compare the "relative gains" from changing heroes, pets or ratios. We recommend calibrating with a real value first.
